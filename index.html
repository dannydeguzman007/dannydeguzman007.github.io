🐙 GitHub Publishing Guide
How to get this site live on the web
▼
1

Create a free GitHub account

Go to github.com → click "Sign up." Pick a username and confirm your email.

2

Create a repository

Click "New". Name it: dannydeguzman007.github.io. Tick "Add a README file" → "Create repository."

3

Upload this file

Click "Add file" → "Upload files." Drag your index.html in and click "Commit changes."

4

Switch on GitHub Pages

Click Settings → "Pages" → choose main → click Save. Wait 2 minutes.

5

Your site is live! 🎉

Visit https://dannydeguzman007.github.io — free forever. Share with the family and bookmark on every phone.

6

To update later

Go to your repository → click index.html → click the pencil ✏️ → edit → "Commit changes." Updates in under a minute.
