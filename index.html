Calendar Reminders

Add your family group email to the shared calendar — then every new event automatically notifies everyone at once. No extra steps needed after that.

Do this once — takes 2 minutes

1. Go to calendar.google.com
2. Find Family Hub in the left sidebar → click the three dots ⋮ → "Settings and sharing"
3. Scroll to "Share with specific people" → click "Add people"
4. Enter deguzman-goose@googlegroups.com
5. Set permission to "See all event details" → click Send

✅ Done! From now on, every new calendar event sends an automatic email to the whole family.

Getting This Online

How to Publish on GitHub

GitHub is a free service where you host this page so everyone in the family can visit it anytime from any phone.

1

Create a free GitHub account

Go to github.com → click "Sign up." Pick a username like ourfamilyhub and confirm your email.

2

Create a repository

Click "New". Name it: yourusername.github.io. Tick "Add a README file" → click "Create repository."

3

Upload this file

Click "Add file" → "Upload files." Drag your index.html in. Write "First upload" and click "Commit changes."

4

Switch on GitHub Pages

Click Settings → "Pages" in the left sidebar → choose main → click Save. Wait 2 minutes.

5

Your site is live! 🎉

Visit https://yourusername.github.io — free forever. Share with the whole family and bookmark on every phone.

6

To update the site (easiest way)

Go to your repository → click "Add file" → "Upload files" → drag the new index.html file onto the page → click "Commit changes." GitHub replaces the old file automatically. Done in under a minute!
